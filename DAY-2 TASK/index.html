Difference between document and window object

Document object

  • The Document object represents the HTML document loaded in the current browser
  • It provides access to the structure and content of the web page
  • It allows to manipulate and interact with its elements

    • Windows object

  • The Window object represents the browser window or tab itself
  • It provides access to controll the browser environment
  • Like opening or closing windows, navigating, timeouts
